# Route-optimizer API

 A simple API that computes an optimized itinerary.


### Installing

Make sure Node.JS is installed

Clone the repo

```
git clone https://github.com/cattode/route-optimizer.git
```

Install dependencies

```
npm install
```

Start the server

```
npm start
```

### Usage

By default, the API runs on `http://localhost:3000/`.



## Running the tests

```
npm test
```
